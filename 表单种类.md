# 表单类型文档

### 文本题

用于输入少量文本，如姓名，邮箱，密码。

#### 题目

```json
{
  "type": "input",
  "data": {
    "label": "",
    "validator": "plain / email / number",
    "remark": "",
    "required": true
  }
}
```

#### 答案

```json
{
  "answer": ""
}
```


### 单选题

用于单项选择。

#### 题目

```json
{
  "type": "radio",
  "data": {
    "label": "",
    "options": ["1", "2", "3"],
    "remark": "",
    "required": true
  }
}
```

#### 答案

```json
{
  "answer": 1
}
```

### 多选题

用于多项选择。

#### 题目

```json
{
  "type": "checkbox",
  "data": {
    "label": "",
    "options": ["1", "2", "3"],
    "maxChoose": 0,
    "other": true,
    "required": true
  }
}
```

#### 答案

```json
{
  "answer": [1, 2, 3]
}
```
